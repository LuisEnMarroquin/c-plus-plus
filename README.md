## Windows Setup

* Install [MinGW Installation Manager](http://www.mingw.org/wiki/Getting_Started)

* Install MinGW and install the following packages in `Basic Setup`
1. mingw32-base-bin
2. mingw32-gcc-g++-bin
3. msys-base-bin

## Compile
```shell
g++ $fileName
```

## Run
```shell
start a.exe
```
